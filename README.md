# __myWayToZenith__

A modern portfolio website showcasing my journey as a Senior Machine Learning Engineer.

## Local Development

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation
```bash
npm install
```

### Running Locally
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Building for Production
```bash
npm run build
```

## Tech Stack
- React 18
- Tailwind CSS
- Framer Motion
- Lucide React Icons
